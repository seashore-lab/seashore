# Development Setup

This repo is an Nx + pnpm monorepo.

## Prerequisites

- Node.js (see root requirements)
- pnpm

## Install

```bash
pnpm install
```

## Build

Build all packages in dependency order:

```bash
pnpm build
```

## Useful commands

```bash
pnpm test
pnpm typecheck
pnpm lint
pnpm format
```
