# Tool Presets

Preset tools are prebuilt integrations shipped with `@seashore/tool`.

There are two categories:

1. **API-key tools** (e.g. Serper / Firecrawl)
2. **Free/public tools** (e.g. DuckDuckGo / Wikipedia / GitHub search)
