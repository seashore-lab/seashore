# GenUI (Generative UI)

GenUI renders tool calls as UI components in a chat interface.

It provides:

- React chat components (`Chat`, `ChatMessages`, `ChatInput`)
- hooks (`useChat`, `useChatStream`)
- a registry system for mapping tool calls to React renderers
