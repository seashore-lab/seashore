# Evaluation

Evaluation helps you measure agent quality and catch regressions.

Seashore supports:

- rule-based metrics
- LLM-based metrics (using a separate evaluator model)
- batch evaluation over datasets

Runnable example: [examples/08-evaluation-qa.md](../examples/08-evaluation-qa.md).
