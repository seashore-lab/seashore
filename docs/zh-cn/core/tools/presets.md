# 工具预设

预设工具是 `@seashorelab/tool` 提供的预构建集成。

有两类：

1. **API 密钥工具**（例如 Serper / Firecrawl）
2. **免费/公共工具**（例如 DuckDuckGo / Wikipedia / GitHub 搜索）

## Serper + Firecrawl

请参阅[示例 11](../../examples/11-tool-presets.md)。

## 新预设工具包

- GitHub 仓库搜索
- 金融报价
- arXiv 论文搜索
- 文章提取
