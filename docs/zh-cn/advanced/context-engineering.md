# 上下文工程

上下文工程是一系列技术，用于在保留重要信息的同时，将提示保持在预算范围内。

这在以下情况下尤为重要：

- 长对话（聊天应用）
- 重工具的智能体
- RAG 上下文 + 对话历史

请参考可运行示例：[examples/14-context-engineering.md](../examples/14-context-engineering.md)。
