# 向量数据库

向量数据库模块 (`@seashorelab/vectordb`) 提供基于 pgvector 的存储和搜索，包括混合搜索（向量 + 全文）。

如果需要可运行的参考，请参见 [examples/13-vectordb-hybrid-search.md](../examples/13-vectordb-hybrid-search.md)。
