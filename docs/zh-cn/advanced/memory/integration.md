# 记忆集成

Seashore 包括将记忆与智能体集成的辅助工具，以便：

- 记忆被添加到系统提示或用户提示中
- 新消息/工具结果自动记录
- 长期回忆可以将相关事实注入上下文

## 中间件和包装器

记忆包导出：

- `withMemory`
- `createMemorySystemPrompt`
- `createMemoryProcessor`
- `createMemoryMiddleware`

当您想要"有记忆的智能体"而不必每次都手动编写提示组装时，请使用这些。

## 简单的手动集成（示例 05 风格）

示例 05 演示了最小方法：

1. 在短期记忆中存储每个用户/助手消息
2. 从记忆构建上下文字符串
3. 将其传递给 `agent.run()`

在添加持久化之前，这是一个很好的基线。
