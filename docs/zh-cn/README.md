# Seashore 文档

Seashore 是一个现代化的 TypeScript 智能体框架，专为构建 AI 驱动的应用程序而设计，具有以下特性：

- 支持 ReAct 风格的智能体及工具调用
- 类型安全的工具定义（Zod）
- 统一的 LLM 适配器（OpenAI / Anthropic / Gemini）
- 用于多步骤编排的工作流
- RAG、内存、存储和向量数据库实用工具
- 生产级关注点：可观测性、评估、安全和部署

如果您是项目的新手，请从这里开始：

- [快速开始](./quick-start.md)
- [运行示例](./introduction.md#运行示例)
