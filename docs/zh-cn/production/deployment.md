# 部署

部署模块 (`@seashore/deploy`) 为 Agent 提供 HTTP 服务器（基于 Hono 构建），具有以下功能：

- 标准聊天和 Agent 端点
- 可选的流式端点
- CORS 和速率限制

可运行示例：[examples/10-deploy-api-server.md](../examples/10-deploy-api-server.md)。
