# 评估

评估可以帮助您衡量 Agent 质量并发现回归问题。

Seashore 支持：

- 基于规则的指标
- 基于 LLM 的指标（使用独立的评估器模型）
- 在数据集上进行批量评估

可运行示例：[examples/08-evaluation-qa.md](../examples/08-evaluation-qa.md)。
