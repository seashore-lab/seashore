# GenUI (生成式 UI)

GenUI 在聊天界面中将工具调用渲染为 UI 组件。

它提供：

- React 聊天组件 (`Chat`、`ChatMessages`、`ChatInput`)
- Hooks (`useChat`、`useChatStream`)
- 用于将工具调用映射到 React 渲染器的注册表系统
